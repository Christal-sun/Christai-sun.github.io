一.多态

  • 什么是多态

    • 多态: 多种形态

    • 多种形态:同一个对象,在不同时刻表现出来的不同形态

    • Cat extends Animal

      • 第一种形态: Cat c1 = new Cat(); //c1是只猫
      • 第二种形态: Animal c2 = new Cat(); //c2是个动物

    • 举例: 数字2

      • 十进制: 2
      • 二进制: 00000010

  • 多态解决程序中的什么问题

    1. 提高代码的复用性

    2. 提高代码的扩展性

  • 多态的使用

    • 多态的代码体现格式

  • 多态在使用时的细节

    1. 多态中父类和子类拥有一模一样的成员变量时:

      • 编译时: 以父类中的成员变量为主
      • 运行时: 使用父类中的成员变量

    2. 多态中父类和子类拥有一模一样的成员方法时:

      • 编译时: 以父类中的成员变量为主(检查父类中有没有这种成员方法,没有就报错)
      • 运行时: 以子类的成员方法为主

  • 多态的转型

    • 向上转型(自动类型提升)

    • 向下转型(强制类型转换)

      • 注意细节

三. Object类

  • 在java语言中，程序员自定义的类或者jdk中存在的类，直接或间接都要继承Object

  • 继承了Object类后，就可以使用Object提供的相关方法：

    • boolean equals(Object obj)

      • boolean： 方法的返回值类型
      • Object： 方法传递参数的类型
      • 作用：比较两个对象是否相同（比较两个对象的地址值是否相同）

八.BigInteger类

  • 可以用来对一些大整数做运算

  • import java.math.BigInteger;

public class BigIntegerDemo {
    public static void main(String[] args) {
        //创建大整数对象
        BigInteger num1 = new BigInteger("1000");
        BigInteger num2 = new BigInteger("2000");

        //运算
        BigInteger result1 = num1.add(num2);
        BigInteger result2 = num1.subtract(num2);
        BigInteger result3 = num1.divide(num2);
        BigInteger result4 = num1.multiply(num2);
        System.out.println(result1);
        System.out.println(result2);
        System.out.println(result3);
        System.out.println(result4);
    }
}
